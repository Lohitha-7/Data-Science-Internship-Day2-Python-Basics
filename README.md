---

### 3. Even or Odd

```python
num = int(input("Enter a number: "))

if num % 2 == 0:
    print("Even")
else:
    print("Odd")
```

---

### 4. For Loop

```python
for i in range(1,6):
    print(i)
```

---

### 5. While Loop

```python
count = 1

while count <= 5:
    print(count)
    count += 1
```

---

### 6. Function

```python
def square(num):
    return num * num

print(square(6))
```
